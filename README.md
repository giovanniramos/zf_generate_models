ZF1 - MODELS GENERATOR
======================

## About ##

Models Generator for the Zend Framework version 1.0.

This script aims to automate the creation of the database models.


## Installation ##

**Skeletons**

    <project name>/
    |    |
    |    application/  
    |    |  
    |    library/  
    |    |  
    |    public/  
    |    |  
    |    scripts/  
    |    |    | 
    +    +    + zf_generate_models/
